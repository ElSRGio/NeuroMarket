✨ Resumen del Sistema

¿Qué ves en las animaciones?

Las animaciones visualizan el proceso completo de NeuroMarket:

  1. ① Entrada: Usuario ingresa datos del negocio (nombre, inversión, ingresos, márgenes, costos)
  2. ② Persistencia: Los datos se guardan en PostgreSQL en tabla analyses
  3. ③ Cálculos: Python Engine ejecuta 5 módulos matemáticos (IRL, TAM/SAM/SOM, ROI, Monte Carlo, SVEE)
  4. ④ Resultado: Se genera Score final, viabilidad y recomendaciones
  5. ⑤ Dashboard: Frontend renderiza resultados con gráficos interactivos

Todas las animaciones muestran este flujo de forma visual y clara.

¿Cómo usar los GIFs?

  • 📊 Presentaciones: Inserta en PowerPoint / Keynote para demostrar el sistema
  • 📱 Redes Sociales: Tweet / Post en LinkedIn mostrando capacidades
  • 📚 Documentación: Agrega a README.md / Wiki técnica
  • 🎓 Educación: Enseña a clientes cómo funciona la plataforma
  • 💼 Propuestas: Incluye en presentaciones de venta
